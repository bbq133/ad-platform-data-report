# AdIntel Growth Scientist 用户操作帮助手册

本文档面向**投放经理、数据科学家、项目负责人**，说明 AdIntel 广告智投分析系统的使用方式与核心功能，帮助您快速上手并发挥系统价值。

---

## 一、产品概述与价值

### 1.1 产品定位

**AdIntel Growth Scientist Suite**（广告智投分析套件）面向**全球众筹与出海品牌**，用于统一分析 Meta 与 Google 广告效果。主要解决：

- **数据归因难**：广告命名不规范，难以按国家、素材类型、漏斗阶段等维度统一分析。
- **多平台对齐难**：Facebook 与 Google 数据格式不一，手工对齐耗时且易错。
- **报表效率低**：从大量素材中筛选高 ROAS 组合、定期产出报表，依赖大量人工。

### 1.2 核心理念

系统将**广告名称中的信息**转化为可分析的**维度**。例如广告名 `US_Prospecting_Video_SpringSale`，可解析出：国家、漏斗阶段、素材类型、活动等，从而支持按这些维度筛选、分组和做透视表报告。

### 1.3 能为您带来的价值

| 角色 | 价值 |
|------|------|
| **投放经理** | 通过 BI 看板与数据透视表快速查看花费、转化、ROAS 等；保存常用报告，一键加载或定时发送。 |
| **数据/分析同学** | 自定义指标公式和维度解析规则，按业务需求做深度分析。 |
| **项目负责人** | 通过定时任务自动接收已保存的透视报告，无需每次手工跑数。 |

## 二、整体使用流程（简要）

1. **登录** → 选择**项目** → **获取广告数据**（选时间、可选账号）。
2. 按需 **调整指标与维度配置**（决定透视表里能看到哪些指标和维度）。
3. 在 **数据透视分析** 中配置行/列/值/筛选器，**保存为报告**，之后可随时加载或**定时发送**。
4. 需要自动报表时，在 **定时任务** 中新建任务，选择已保存的报告、发送频率和收件人即可。

下面按操作顺序说明各环节。

---

## 三、登录与项目选择

### 3.1 登录

1. 打开系统后，若未登录会进入**登录页**。
2. 输入**用户名**和**授权密码**，点击「身份认证」。
3. 验证通过后进入**项目选择**页；失败会提示错误信息。

### 3.2 选择项目

1. 可用顶部搜索框**按项目名称搜索**。
2. 点击某个**项目卡片**即进入该项目的分析工作台。
3. 若要换项目，可通过数据源配置中的项目选择器或返回项目选择页重新选。

---

## 四、获取广告数据

### 4.1 配置数据源

进入项目后，若尚未加载数据，会先进入**配置数据源**页面：

1. **选择项目**：确认或更换当前项目。
2. **选择时间范围**：选择开始日期和结束日期（以界面提示的可选范围为准）。
3. **选择账号（可选）**：若项目下有多个广告账号，可勾选需要分析的账号；不选则拉取全部。

点击 **「获取广告数据」** 后，系统会**同时拉取 Facebook 与 Google** 的数据；加载完成后会自动进入分析界面。

### 4.2 获取数据后的筛选

数据加载完成后，在**仪表盘**顶部有**全局筛选**：

- **日期范围**：可调整分析的时间段。
- **平台**：可限定为 Meta、Google 或全部。
- **刷新**：需要最新数据时，可点击刷新重新拉取。

后续的 BI 看板、数据透视表都会基于当前选中的日期和平台。

## 五、指标与维度配置（重点）

指标与维度配置决定了：**透视表里有哪些维度可拖拽**、**有哪些指标可汇总**、以及 **BI 看板展示什么**。建议在开始做透视报告前先完成基础配置。

### 5.1 如何进入配置

- 在已加载数据的仪表盘页面，点击顶部 **「调整指标配置」** 进入配置模块。
- 修改后务必点击 **「【保存指标与维度参数配置】」**，配置才会生效并在下次打开时保留。

### 5.2 指标与公式

- **作用**：定义系统中有哪些可用的**指标**（包括计算指标）。
- **基础指标**：如花费、展示、点击、转化等，由系统根据数据源提供。
- **自定义公式**：您可以新增计算指标，例如：
  - 单次点击成本：`花费 / 链接点击数`
  - ROAS：`转化金额 / 花费`
- 可为每个指标设置**单位**（如金额用「$」，比率用「%」）。
- 公式中引用的字段必须来自当前数据中已有的字段，保存时系统会做校验。

配置好后，这些指标会在 **数据透视表的值字段** 和 **BI 看板** 中可选。

### 5.3 维度参数配置（命名规范解析）

- **作用**：把广告名称（Campaign 名称、Ad Set 名称、Ad 名称等）拆成可分析的**维度**。
- **典型用法**：  
  若广告命名类似 `US_Prospecting_Video_SpringSale`（用下划线 `_` 分隔），您可以：
  - 设置**分隔符**为 `_`；
  - 指定**第 1 段**对应「国家」、**第 2 段**对应「漏斗」、**第 3 段**对应「素材类型」等。
- 配置完成后，这些维度会出现在**数据透视表的行/列/筛选器**中，供您自由组合。
- 每个维度可以指定数据来源（来自 Campaign 名称、Ad Set 名称还是 Ad 名称），以及取「整段」还是「第 N 段」。

这样，即使平台导出的原始列里没有「国家」「漏斗」等，您也能通过命名规范解析出这些维度做分析。

### 5.4 数据质量报告

- 用于在正式分析前**快速查看**当前数据的质量情况（如缺失、异常等）。
- 查看完毕后，可点击「生成智投分析面板」返回仪表盘。

### 5.5 小结

- **指标与公式** → 决定「能算哪些数」；
- **维度参数配置** → 决定「能按哪些维度拆解」；
- 两者配置好并**保存**后，数据透视表和 BI 看板中才会有完整的指标与维度可选。

## 六、数据透视表与报告（重点）

数据透视分析用于**按维度与指标自由组合**，生成表格报告，并可**保存、重复使用、定时发送**。

### 6.1 透视表是什么

- **行**：作为行展示的维度（如国家、Campaign、素材类型）。
- **列**：作为列展示的维度（可选，用于交叉分析）。
- **值**：要汇总的指标（如花费、点击、ROAS）。
- **筛选器**：对某些维度做条件筛选（如只选某几个国家、某段时间）。
- **平台范围**：选择数据来自哪些平台或广告类型（如仅 Meta、仅 Google 搜索等）。

您通过「字段配置」选择行、列、值和筛选器，表格会实时按您的选择汇总数据。

### 6.2 字段配置

- 点击 **「字段配置」** 打开配置面板。
- 可进行：
  - **行**：添加/删除/拖拽排序行维度。
  - **列**：添加/删除/拖拽排序列维度。
  - **值**：添加/删除要汇总的指标。
  - **筛选器**：添加筛选条件（多选、包含/不包含、日期范围等）。
  - **显示选项**：如是否显示小计、总计等。
  - **平台范围**：勾选参与透视的平台或类型。
- 配置会即时反映在下方的透视表中。

### 6.3 保存报告

- 当行/列/值/筛选器/平台范围都配置好后，点击 **「【保存为新的报告】」**。
- 输入**报告名称**后即可保存。  
- 保存后，该报告会出现在 **「已保存报告」** 下拉列表中，可在本机或同账号下再次加载使用。

### 6.4 加载已保存报告

- 点击 **「已保存报告」** 下拉，选择某条报告。
- 系统会将该报告的**全部配置**（行/列/值/筛选器/平台范围等）加载到当前页面，并在顶部显示「当前报告：报告名称」。
- 加载后您可以在此基础上微调，再选择「更新当前报告」或「保存为新的报告」。

### 6.5 更新当前报告

- 当您**已加载**某条报告并修改了字段配置后，若希望用当前配置**覆盖**这条报告（名称不变）：
  - 可点击 **「更新当前报告」**，或  
  - 在「已保存报告」下拉里，将鼠标悬停在该报告上，点击 ✏️ 图标。
- 确认后，该报告即更新为当前配置，无需另存为新报告。

### 6.6 删除报告

- 在「已保存报告」下拉中，将鼠标悬停到要删除的报告上，点击 🗑️ 图标。
- 按提示确认后即可删除。删除后该报告不再出现在列表中，也无法被定时任务选用。

### 6.7 导出

- 透视表支持**导出为 Excel**，便于在本地加工或分享给他人。

### 6.8 报告与账号

- 已保存报告按**当前登录用户、所选项目及账号**区分；切换项目或账号后，看到的报告列表会对应变化，互不混淆。

## 七、定时发送已保存报告

系统支持在指定时间**自动生成并发送**您已保存的透视报告到邮箱，无需每次手工跑数、发邮件。

### 7.1 入口

- 在项目内，点击顶层 **「定时任务」** Tab，进入定时任务页面。

### 7.2 定时任务列表与发送历史

- **定时任务**：列出您已创建的所有定时任务（名称、频率、时间、收件人、关联报告等）。
- **历史**：可查看每次任务是否执行成功、发送时间、收件人及失败时的提示，便于排查问题。

### 7.3 新建定时任务

点击 **「新建定时任务」**，在表单中填写：

- **任务名称**：便于识别，如「每周国家效果报表」。
- **频率**：  
  - **每日**：每天在指定时间执行一次。  
  - **每周**：需再选择星期几（如每周一）。
- **执行时间**：一天中的具体时刻（如 09:00）。
- **日期范围**：报告数据覆盖的时间段，如「近 7 天」「近 15 天」「近 30 天」，或自定义起止日期。
- **要发送的报告**：从**已保存报告**中勾选一个或多个；到点后会按这些报告的配置生成数据并发送。
- **收件邮箱**：填写一个或多个邮箱地址，多个可用逗号分隔。
- **是否启用**：可先关闭，需要时再开启。

保存后，任务会按您设置的频率和时间自动执行；执行时会用您选的「日期范围」拉取数据，按所选报告的配置生成透视表并发送到指定邮箱。

### 7.4 编辑、启用/停用、删除

- **编辑**：在任务列表中点击编辑，可修改频率、时间、报告、收件人等，保存即可。
- **启用/停用**：可暂时停用某条任务，不删除也不影响历史记录；需要时再启用。
- **删除**：删除后该任务不再执行，列表中不再显示。

## 八、BI 看板（简要）

在「数据分析」下，除数据透视分析外，还有 **BI 看板**：

- 展示 **KPI 卡片**（如总花费、总 Leads、平均 CPL 等）及与上一周期的对比。
- 提供**平台**与 **Google 广告类型**的切片器，可快速切换查看不同平台或类型的数据。
- 提供**核心指标趋势图**，可多选指标查看走势。

BI 看板与数据透视表共用同一套全局筛选（日期、平台）；指标来源由「指标与维度配置」决定。

## 十、其他实用说明

### 10.1 顶部常用按钮

- **调整指标配置**：在仪表盘页点击，进入指标与维度配置。
- **【保存指标与维度参数配置】**：在配置页点击，保存当前配置，下次打开或换设备会沿用。
- **用户与登出**：右上角可看到当前用户信息，点击登出图标可退出登录。

### 10.2 配置与数据的保存与同步

- **指标与维度配置**：保存后会在您的账号下同步，下次进入同一项目时会自动加载。
- **已保存报告**：保存、更新或删除后都会同步到您的账号下，同账号同项目下可见。
- **定时任务**：保存后即按设定执行；若执行失败，可在「历史」中查看原因。
- **原始广告数据**：每次打开页面后需通过「获取广告数据」拉取；刷新页面后如需最新数据可再次点击获取或使用顶部刷新。

### 10.3 常见问题

- **透视表里看不到某个维度或指标**：请到「调整指标配置」中检查是否已在「维度参数配置」里添加该维度、在「指标与公式」里添加或映射该指标，并完成保存。
- **定时任务没有收到邮件**：先到「定时任务」→「历史」查看该任务是否有执行记录及状态；若为失败，可根据提示信息排查（如邮箱是否填写正确）。
- **在不同电脑或浏览器里报告不一致**：请确认使用同一账号登录、同一项目；报告列表按用户与项目区分，需在相同环境下保存与加载。

## 十一、总结

| 环节 | 要点 |
|------|------|
| **登录与项目** | 用户名与密码登录，选择项目进入工作台。 |
| **数据源** | 选择项目、时间、账号后点击「获取广告数据」，拉取 Meta 与 Google 数据。 |
| **指标与维度配置** | 在「调整指标配置」中设置指标公式与维度解析规则，保存后透视表与 BI 看板才有完整的指标与维度可用。 |
| **数据透视与报告** | 配置行/列/值/筛选器，保存为报告；可随时加载、更新或删除报告，并导出 Excel。 |
| **定时任务** | 在「定时任务」中新建任务，选择已保存的报告、发送频率、时间和收件人，即可自动收到报表。 |
| **BI 看板** | 查看 KPI 与趋势图，配合切片器按平台与类型查看。 |

用好**指标与维度配置**和**数据透视报告**的保存与定时发送，即可大幅提升日常报表效率，减少重复操作。
