# 📊 快速上手：更新数据透视报告

## 🎯 3 步更新报告配置

### 步骤 1：加载报告
点击"已保存报告" → 选择要修改的报告

### 步骤 2：修改配置
点击"字段配置" → 调整行/列/值/筛选器

### 步骤 3：保存更新
打开"已保存报告" → 鼠标悬停 → 点击 ✏️ 图标 → 确认

---

## 💡 功能亮点

✅ **一键更新** - 无需重新保存
✅ **保持名称** - 报告名称不变
✅ **自动同步** - 立即同步到 Google Sheets
✅ **操作安全** - 更新前确认，防止误操作

---

## 🔧 可更新的配置

- 筛选器（添加/删除/修改条件）
- 行字段（拖动排序/添加/删除）
- 列字段（拖动排序/添加/删除）
- 值字段（添加/删除指标）
- 平台范围（选择 Meta/Google 等）
- 显示选项（小计/总计设置）

---

## 📝 操作提示

- **更新按钮**：鼠标悬停在报告名称上显示 ✏️ 图标
- **删除按钮**：同时显示 🗑️ 图标
- **确认提示**：更新前会显示报告名称，避免误操作
- **成功提示**：更新后显示"已更新并同步到云端 ✅"

---

## ⚡ 示例场景

**场景：为"每日国家分析"报告添加新的筛选器**

1. 点击"已保存报告" → 选择"每日国家分析"
2. 点击"字段配置" → 在"筛选器"区域添加"广告类型"筛选
3. 返回"已保存报告" → 悬停在"每日国家分析"上 → 点击 ✏️
4. 确认更新 → 完成！
