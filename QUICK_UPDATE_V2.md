
## 🚀 快速上手

### 编辑已有报告
```
1. 加载报告 → 看到"当前报告"标签
2. 修改配置 → 调整字段
3. 点击"更新当前报告" → 完成！
```

### 创建新报告
```
1. 点击"新建报告"（如有活动报告）
2. 配置字段
3. 点击"保存为报告" → 完成！
```
